# python3, Pyqt5 ,
## python pyqt5 的仿kfc点餐系统

## 项目安装

```
$ git clone https://github.com/fanyaow/kfc.git
$ cd kfc
$ pip install -r requirements.txt
```
## 运行项目
```
pycharm中打开kfc.py， 右键 run 'kfc'
```
